# 💼 Pharmaceutical Portfolio Website

A modern, responsive portfolio website designed and developed for a pharmacy student to showcase academic achievements, clinical training, certifications, skills, and professional experience.

## ✨ Highlights

* Modern UI/UX Design
* Fully Responsive Layout
* Advanced Animations
* Interactive Components
* Professional Portfolio Structure
* CV Integration
* Contact System

## 🛠️ Built With

* HTML5
* CSS3
* Tailwind CSS
* JavaScript
* GSAP
* Lucide Icons

## 🎯 Project Goal

To create a premium, internship-focused portfolio that presents academic and professional information in a modern and engaging way.
